Color Scheme

Carolina Blue (279AF1) will work nicely for the header and footer elements while Ghost White (F7F7FF) will work nice for the main content background. If there is the need for any third color, we would use either Pacific Blue (23B5D3) if we want the color to complement Carolina Blue or Paradise Pink (EA526F) if we wanted the colors to contrast. Rich Black Fogra 39 (070600) would be suitable for any borders or shadows we wished to implement.
